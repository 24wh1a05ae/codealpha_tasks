# 🎵 CodeAlpha Music Player

A simple and responsive **Music Player** built using **HTML, CSS, and JavaScript** as part of the **CodeAlpha Web Development Internship**. The application provides an intuitive interface to play, pause, and navigate through songs with a clean and modern design.

## 🚀 Features

* ▶️ Play and Pause music
* ⏭️ Next and Previous song navigation
* 🎶 Display song title and artist
* 📊 Interactive progress bar
* 🔊 Volume control
* 📱 Responsive user interface
* 🎨 Modern and user-friendly design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it using a web browser.
4. Enjoy listening to your music!
